# CONTROLE DE NOTAS			
	Versão 1.0

Obrigado por utilizar esse sistema! Foi feito com muito carinho pensando em todos os usuários!
Essa é a primeira versão, no futuro serçao lançadas atualizações de interface e novas funcionalidades como controle de faltas!
Espero que possamos nos encontrar em Breve :)

Este aplicativo permite gerenciar:
- Cadastro de alunos
- Registro de tarefas
- Boletim por trimestre
- Boletim Final

REQUISITOS:
- Sistema Windows 64 bits
- Conexão com Internet

AVISOS E RECOMENDAÇÕES:
1. Siga as instruções do "GUIA DE INSTALAÇÃO"
2. Nunca separe o programa de sua pasta de origem
3. Guarde a pasta em um lugar de fácil acesso
4. Não perca sua senha, o programa possui um complexo sistema de senhas que nem mesmo eu vou saber sua senha.
5. Ao rodar o programa, talvez apresente certa instabilidade dependendo de sua conexão com a internet, avisos como: "Server reset" ou coisas parecidas podem ser normais
Em casos como esse, apenas deem ok na mensagem e tentem realizar a ação que estavam tentando novamente. Em caso de persistência no erro, me contate ou tente reiniciar o app.
6. Comandos como ctrl + c/ ctrl + v são mal interpretados pelo sistema, o que pode ocasionar algum erro como "Usuário ou senha inválidos" mesmo não estando
7. Caso você encontre algum erro ou dificuldade me contate via E-mail (rafinhafiorioofc@gmail.com)
8. Só relembrando novamente que esse é um projeto desenvolvido apenas por mim com um gasto mínimo o que não permite grande velocidade ou altas verificações :)
9. Ultimo aviso, prometo hehehe. Caso você esteja lendo isso via github, não disponibilizei o pacote por ali por ser algo pra uso mais pessoal. Obrigado por entender
